Trabalhando com Listas

Listas Ordenadas

Utilizando a tag "ol", refere-se a Lista Ordenada. Utilizando a tag "li", faz-se uma ordem como 1,2,3... A,B,C... I,II,III...porém é necessário utilizar a função "type" (1, A, a, I, i).

  1. Acordar
  2. Levar Brenda ao Trabalho
  3. Tomar café
  4. Escovar os dentes
  5. Arrumar cama

Listas Não Ordenadas

Utilizando a tag "ul", refere-se a Lista Não Ordenada. Utilizando a tag "li", referencia os itens da lista. Formas de marcação "type" (disc, circle, square).

  • Pao
  • Leite
  • Ovo
  • Farinha
  • Arroz
  • Feijão
